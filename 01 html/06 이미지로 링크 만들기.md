# 6. 이미지로 링크 만들기
- `<img>` 태그를 `<a>` 태그로 감싸서 링크를 만듭니다. 
  
![이미지로 링크 만들기](img/06.png)
```html
<!DOCTYPE html>
<html lang="ko">

<head>
  <meta charset="UTF-8">
  <meta name="description" content="이미지 링크를 만듭니다.">
  <title>이미지 링크 만들기</title>
</head>

<body>
  <a href="https://www.google.co.kr/"><img src="img/christmas.jpg" alt="아름다운 벚꽃"></a>
</body>

</html>
```
